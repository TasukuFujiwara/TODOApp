# Features
* TODOの作成・編集・削除
* フォルダを作成し，TODOをグループ化できる
* 使いやすくてシンプルなUI（画面のデザインは，iPhoneの標準メモアプリとTwitterをイメージ）
* 作成したTODOは，期日順に自動で並び替えられる
* TODO,フォルダはデータベースで管理されている


# Installation
